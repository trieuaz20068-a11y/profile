Pen&Paper — Website mẫu bán dụng cụ học tập

Nội dung:
- `index.html` — trang chính (tĩnh)
- `styles.css` — CSS cho layout và responsive
- `script.js` — JS nhỏ quản lý giỏ hàng demo và lọc sản phẩm

Cách chạy:
1. Mở file `d:\Bài kiểm tra\bài cuối môn\stationery-store\index.html` bằng trình duyệt.
2. Thêm sản phẩm vào giỏ bằng nút "Thêm vào giỏ". Nhấn biểu tượng giỏ để mở giỏ hàng.

Ghi chú
- Ảnh sản phẩm dùng placeholder/gradient. Bạn có thể thay bằng ảnh thực tế (đặt file vào cùng thư mục và sửa `products` data trong `script.js`).
- Tính năng thanh toán chỉ là demo (alert).

Muốn mở rộng?
- Thêm trang chi tiết sản phẩm, tìm kiếm nâng cao, bộ lọc giá, lưu giỏ vào localStorage, hoặc kết nối backend.
